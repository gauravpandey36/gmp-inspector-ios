# GMP Inspector iOS App

AI-powered GMP floor inspection for pharmaceutical facilities.
Meta Ray-Ban + Claude AI (images) + Gemini AI (video).

## Features
- **Photo Analysis** (Claude): Snap a photo, get instant GMP compliance report
- **Video Analysis** (Gemini): Record a walkthrough, AI detects behavioral violations
- **Voice Alerts**: Critical findings read aloud through earbuds
- **4 Inspection Skills**: Auto-Detect, QA Floor Ops, EHS Safety, GDP
- **8 Compliance Domains**: Area Status, Equipment, GDP, Gowning, Materials, Environmental, EHS, Line Clearance

## Build
Built with Flutter. iOS builds via Codemagic CI/CD.
